Sobre mí


Winder Delgado ¡Hola! Soy Winder Delgado, Técnico Superior en Informática, Programador y Desarrollador Web, actualmente me desenvuelvo como Desarrollador Web, con una pasión por crear soluciones backend robustas y eficientes. Aunque mi fortaleza principal reside en el desarrollo backend, también tengo conocimientos sólidos en HTML y CSS, y manejo básico de JavaScript para complementar mis habilidades frontend. Desde que comencé mi viaje en el mundo del desarrollo web, he estado inmerso en el aprendizaje y la exploración de las últimas tecnologías y prácticas en backend. Como Desarrollador Web también tengo experiencia en programación de escritorio utilizando Tkinter, además de conocimientos básicos en data science, machine learning y deep learning con Python. Tengo un sólido entendimiento de Python y he trabajado con frameworks como Django, Flask y FastAPI para la creación de APIs con Jinja2, y manejo de templates y API REST para proporcionar datos en formato JSON. También poseo conocimientos básicos de SQL y experiencia en el manejo de bases de datos relacionales como MySQL y PostgreSQL. En el ámbito de FrontEnd como desarrollo web, manejo HTML, CSS y JavaScript puro (Vanilla JavaScript), y utilizo GitHub para el control de versiones. Estoy emocionado por aplicar mis conocimientos para desarrollar interfaces de usuario intuitivas y atractivas, así como soluciones robustas y eficientes. Mi objetivo es crear experiencias de usuario que no solo sean visualmente impactantes, sino también altamente funcionales y accesibles. Me apasiona resolver problemas complejos con soluciones tecnológicas innovadoras, optimizar procesos y mejorar la eficiencia en el desarrollo de software. Estoy comprometido en mantenerme actualizado con las últimas tendencias y tecnologías del sector para ofrecer siempre lo mejor en cada proyecto.
